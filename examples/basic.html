Zero WYSIWYG — Basic Example

Full-featured editor with slash commands enabled, word count, and all built-in features. English, light theme.

✨ Try these features:

• Slash commands: Type / on a new line to see the command menu
• Find & Replace: Press Ctrl+F to search within your content
• Task lists: Click the checkbox icon ☑ in the toolbar, then check/uncheck items
• Code blocks: Click the </> icon to insert a code block with language selector
• Table editing: Insert a table, then right-click any cell for add/delete row/column options
• Emoji picker: Click the 😀 icon in the toolbar
• Color picker: Text color and background color with 40-swatch palette
