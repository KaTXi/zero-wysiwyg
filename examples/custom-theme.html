Zero WYSIWYG — Custom Themes

Create your own theme by overriding --zw-* CSS variables on a parent element. No build step needed.

🟣 Purple Theme (Light)

🟢 GitHub-style Dark Theme

How it works

/* Just override CSS variables on a parent element */
.my-custom-theme {
    --zw-accent: #8b5cf6;
    --zw-accent-hover: #7c3aed;
    --zw-accent-light: rgba(139, 92, 246, 0.1);
    --zw-bg-toolbar: #faf5ff;
    --zw-border: #e9d5ff;
}

/* Then wrap your textarea */
<div class="my-custom-theme">
    <textarea id="editor"></textarea>
</div>
