Zero WYSIWYG — Markdown Mode

Bidirectional HTML ↔ Markdown conversion. Write in the WYSIWYG editor, export Markdown — or paste Markdown and import it as rich HTML.

📝 Markdown API

• ZeroWysiwyg.getMarkdown('id') — Export the editor's HTML content as Markdown
• ZeroWysiwyg.setMarkdown('id', md) — Import Markdown into the editor as rich HTML
• Supports: headings, bold, italic, strikethrough, links, images, lists, task lists, blockquotes, code blocks, inline code, tables, and horizontal rules
📄 Markdown (editable — click "Import" to load into editor)
🔤 HTML Output (read-only)
