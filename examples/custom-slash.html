Zero WYSIWYG — Custom Slash Commands

Add your own commands to the / slash menu using the slashItems option. Custom items appear alongside the built-in commands.

⚡ How it works

• Set slashCommands: true to enable the slash menu
• Add slashItems: [...] with your custom commands
• Each item needs: label, icon (emoji/text), description, and an action(editor) callback
• Custom items appear above the built-in commands in the menu
• Type / on an empty line or at the start of a line to open the menu
// Custom slash items configuration
slashItems: [
  {
    label: 'Alert Box',
    icon: '🚨',
    description: 'Insert a warning alert',
    action: function(editor) {
      editor.insertHTML('<div style="...">⚠️ Alert</div>');
    }
  }
]
