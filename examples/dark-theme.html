Dark theme, Spanish locale, balloon toolbar mode (toolbarMode: 'both'), slash commands, and word count. Select text to see the floating toolbar!

• Listas de tareas: Usa el botón ☑ del toolbar

• Navegación de tablas: Pulsa Tab / Shift+Tab para moverte entre celdas
• Emojis: 320+ emojis integrados con búsqueda
• Auto-enlace: Pega una URL y se convierte en enlace automáticamente
• Markdown: Exporta el contenido como Markdown con ZeroWysiwyg.getMarkdown()
