Zero WYSIWYG — Multi-Locale Example

Multiple editors on the same page, each with its own language. The toolbar labels, dialogs, slash commands, and word count all use the per-instance locale.

🌐 Per-Instance Locale (v1.2)

• Each editor gets its own locale option — they don't share a global language
• The word count bar shows localized labels: "107 words" vs "107 palabras" vs "107 mots"
• Toolbar tooltips, dialogs, slash command labels — all follow the per-instance locale
• External locale files (src/i18n/*.js) must be loaded via <script> for languages beyond EN/ES
